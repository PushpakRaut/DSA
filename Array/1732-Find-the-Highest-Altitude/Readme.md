<h4>Question</h4>
  <h2>Problem Statement:</h2>
  <p>
    The biker is going on a road trip, and the road trip consists of n + 1 points at different altitudes. The biker starts the trip at point 0 with an altitude of 0. The net gain in altitude between each pair of consecutive points is given by the array 'gain' of length n.
  </p>

  <h3>Example 1:</h3>
  <p>
    <strong>Input:</strong> gain = [-5, 1, 5, 0, -7]<br>
    <strong>Output:</strong> 1<br>
    <strong>Explanation:</strong> The altitudes are [0, -5, -4, 1, 1, -6]. The highest altitude is 1.
  </p>

  <h3>Example 2:</h3>
  <p>
    <strong>Input:</strong> gain = [-4, -3, -2, -1, 4, 3, 2]<br>
    <strong>Output:</strong> 0<br>
    <strong>Explanation:</strong> The altitudes are [0, -4, -7, -9, -10, -6, -3, -1]. The highest altitude is 0.
  </p>

  <h3>Constraints:</h3>
  <ul>
    <li>n == gain.length</li>
    <li>1 ≤ n ≤ 100</li>
    <li>-100 ≤ gain[i] ≤ 100</li>
  </ul>
