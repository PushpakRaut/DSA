<h4>Question</h4>
    <p>Given an integer <code>n</code>, return any array containing <code>n</code> unique integers such that they add up to 0.</p>
    <h2>Examples:</h2>
    <h3>Example 1:</h3>
    <pre>
    Input: n = 5
    Output: [-7,-1,1,3,4]
    Explanation: These arrays also are accepted [-5,-1,1,2,3] , [-3,-1,2,-2,4].
    </pre>
    <h3>Example 2:</h3>
    <pre>
    Input: n = 3
    Output: [-1,0,1]
    </pre>
    <h3>Example 3:</h3>
    <pre>
    Input: n = 1
    Output: [0]
    </pre>
    <h2>Constraints:</h2>
    <ul>
        <li>1 <= <code>n</code> <= 1000</li>
    </ul>
