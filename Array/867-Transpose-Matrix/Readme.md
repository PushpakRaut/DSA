<h4>Question</h4>
    <p>Given a 2D integer array <code>matrix</code>, return the transpose of matrix.</p>
    <p>The transpose of a matrix is the matrix flipped over its main diagonal, switching the matrix's row and column indices.</p>
    <h2>Examples:</h2>
    <h3>Example 1:</h3>
    <pre>
    Input: matrix = [[1,2,3],[4,5,6],[7,8,9]]
    Output: [[1,4,7],[2,5,8],[3,6,9]]
    </pre>
    <h3>Example 2:</h3>
    <pre>
    Input: matrix = [[1,2,3],[4,5,6]]
    Output: [[1,4],[2,5],[3,6]]
    </pre>
    <h2>Constraints:</h2>
    <ul>
        <li><code>m</code> == <code>matrix.length</code></li>
        <li><code>n</code> == <code>matrix[i].length</code></li>
        <li>1 <= <code>m</code>, <code>n</code> <= 1000</li>
        <li>1 <= <code>m * n</code> <= 10^5</li>
        <li>-10^9 <= <code>matrix[i][j]</code> <= 10^9</li>
    </ul>
