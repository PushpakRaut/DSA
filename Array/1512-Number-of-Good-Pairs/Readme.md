<h4>Question</h4>
    <p>Given an array of integers <code>nums</code>, return the number of good pairs.</p>
    <p>A pair (<em>i, j</em>) is called good if <code>nums[i] == nums[j]</code> and <em>i</em> &lt; <em>j</em>.</p> 
    <h3>Example 1:</h3>
    <p>Input: nums = [1,2,3,1,1,3]</p>
    <p>Output: 4</p>
    <p>Explanation: There are 4 good pairs (0,3), (0,4), (3,4), (2,5) 0-indexed.</p>   
    <h3>Example 2:</h3>
    <p>Input: nums = [1,1,1,1]</p>
    <p>Output: 6</p>
    <p>Explanation: Each pair in the array is good.</p>  
    <h3>Example 3:</h3>
    <p>Input: nums = [1,2,3]</p>
    <p>Output: 0</p>  
    <h3>Constraints:</h3>
    <ul>
        <li>1 &le; nums.length &le; 100</li>
        <li>1 &le; nums[i] &le; 100</li>
    </ul>
