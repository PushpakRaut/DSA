<h4>Question</h4>
    <p>Given a positive integer <code>n</code>, generate an n x n matrix filled with elements from 1 to <code>n^2</code> in spiral order.</p>
    <h2>Examples:</h2
    <h3>Example 1:</h3>
    <pre>
    Input: n = 3
    Output: [[1,2,3],[8,9,4],[7,6,5]]
    </pre>
    <h3>Example 2:</h3>
    <pre>
    Input: n = 1
    Output: [[1]]
    </pre>
    <h2>Constraints:</h2>
    <ul>
        <li>1 <= <code>n</code> <= 20</li>
    </ul>
